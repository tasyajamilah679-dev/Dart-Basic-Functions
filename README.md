# Dart Basic Functions

## Deskripsi Project
Project ini berisi implementasi dasar function dalam bahasa Dart dengan 3 topik:
1. Menyapa
2. Kalkulator Luas Persegi
3. Kalkulator Diskon

Setiap topik memiliki 4 jenis function:
- Tanpa parameter & tanpa return
- Dengan parameter tanpa return
- Tanpa parameter dengan return
- Dengan parameter & return

---


### 1. Function Tanpa Parameter & Tanpa Return
Function ini tidak menerima input dan tidak mengembalikan nilai.
Contoh:
```dart
void sapa1() {
  print("Halo!");
}
